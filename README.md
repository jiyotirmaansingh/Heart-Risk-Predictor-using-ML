🫀 Heart Risk Predictor using Machine Learning

Heart disease remains one of the leading causes of mortality worldwide. Early detection and preventive care play a crucial role in reducing severe outcomes. This project analyzes clinical health indicators and builds a machine learning model to predict the likelihood of heart disease.

The system combines data preprocessing, exploratory analysis, model comparison, and an interactive prediction interface to support healthcare awareness and decision-making.
